## How to run:

Install dependencies:
```bash
pip install streamlit geopandas pandas requests shapely matplotlib pydantic
```

Launch the app:
```bash
streamlit run app/streamlit_app.py
```

The app will download all datasets automatically and open in the browser.

To run the tests:
```bash
pytest
```
